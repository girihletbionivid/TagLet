Features

Featur

Raw Data Quality Control

User friendly GUI for performing quality control of NGS Raw Data across the platform. NGS-QC-toolkit has been used to process the Raw Data.

RNA-Seq Denovo

User friendly GUI for performing denovo assembly of RNA-Seq data. The process invovles sample merging, assembly, validatation and reporting of RNA-Seq assembly. Trinity, Samtools, Bowtie and various In-house developed scripts have been used to process the quality controlled read.

RNA-Seq Differential Analysis

User friendly GUI for performing differential expression profilling of RNA-Seq data, with automated reporting. DESeq has been used to process the data.

Metagenome 16s RNA Analysis

User friendly GUI for performing comprehensive analysis and automated reporting of 16s RNA amplicon sequencing. The processes involves Read Prepration, Dereplication, Clusturing, OTU identification & Diversity Analysis using Taxonomic Profiling. UPARSE tool has been used for processing of data.
